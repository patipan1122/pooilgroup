Animations — สำหรับ preview (LINE จริงเป็นภาพ static)
บน HTML preview (ที่คุณเห็นนี้)
  • · float-y · ลอยขึ้น-ลง 3.6s — บนการ์ดด้านบน banner
  • · bob · เด้งซ้าย-ขวา — บน character + primary icons
  • · wiggle · บิดเล็กน้อย — บน icon urgent (ซ่อม, แจ้ง)
  • · sparkle · กระพริบ scale — บนดาว/ประกาย
  • · shine · sweep gradient — บนปุ่ม primary
  • · hover lift · ทุกปุ่มยกขึ้น 3px + icon scale
บน LINE จริง

Rich menu = static image (PNG/JPEG) ขึ้นบน LINE — ขยับ animation ไม่ได้. แต่ feel "มีชีวิต" ได้จาก:

  • 1. Animated GIF — LINE รองรับ GIF rich menu (โหลดช้ากว่า)
  • 2. Switch รูป ตามช่วงเวลา · ผ่าน push update
  • 3. ใส่ shine/sparkle ใน image ตายตัว — ยัง feel ดูใหม่
