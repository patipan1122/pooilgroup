# คู่มือใช้งาน — Branch Manager / Admin

> สำหรับ: ผู้จัดการสาขา · หัวหน้าเขต · พนักงานออฟฟิศที่ดูแลข้อมูล · ผู้ที่มีตำแหน่งในระบบเป็น `admin` · `area_manager` · หรือ `branch_manager`
> Last updated: 2026-05-20

---

## 📌 1. ภาพรวม

คุณคือ "คนตรงกลาง" ระหว่าง Owner กับพนักงานสาขา · มีหน้าที่ดูแลและตรวจสอบรายงาน

ในมุมระบบ คุณ:

- ดูยอดขายของ **สาขาที่คุณดูแล** (1 สาขา · หลายสาขาในเขต · หรือทุกสาขาถ้าเป็น `admin`)
- **อนุมัติรายงานประจำวัน** ของสาขาที่ดูแล
- กรอกรายงานแทนสาขาที่ลืมส่งได้
- ดูเงินขาด · Health Score ของสาขา
- เห็น Dashboard ภาพรวม
- **ไม่สามารถ:** ปลดล็อกรายงานที่อนุมัติแล้ว (สิทธิ์เฉพาะ super_admin)

> 💡 ความต่าง 3 ตำแหน่งนี้:
> - `branch_manager` = ดูแล 1 สาขา · อนุมัติของสาขาตัวเองเท่านั้น
> - `area_manager` = ดูแลหลายสาขาในเขต · กรอกแทนสาขาในเขตได้
> - `admin` = ออฟฟิศกลาง · อนุมัติได้ทุกสาขา · เชิญพนักงานได้

## ✅ 3. งานประจำวัน (Daily Tasks)

### 3.1 ตรวจรายงานที่รออนุมัติ (ทุกเช้า)

**ทำอะไร:** ตรวจรายงานเมื่อวานก่อนปิดบัญชี

1. เข้า `/home` — ดู section "รายงานรออนุมัติ"
2. หรือเข้า `/cashhub/reports` → กรอง chip "รออนุมัติ"
3. กดเข้ารายงานทีละใบ
4. ตรวจ:
   - **ยอดรวม** ตรงกับเป้า / ตัวเลขเมื่อวานที่ใกล้เคียงไหม
   - **Reconcile** ตรงกัน (ยอดในระบบ vs เงินจริง)
   - **เงินขาด** ถ้ามี — ดูว่ามีระบุคนรับผิดชอบ + เหตุผลครบ
   - **เวลา** ส่งก่อน deadline (default 21:00) หรือเปล่า
5. ถ้าถูก → กด **อนุมัติ** (สีเขียว)
6. ถ้าผิด → กด **ไม่อนุมัติ** + ใส่เหตุผล → ระบบส่งกลับให้สาขาแก้

> 💡 **Quick Approve Bar:** ถ้ามีหลายใบที่มั่นใจว่าถูก กดเลือกหลายใบแล้วกด "อนุมัติทั้งหมด"

[screenshot: หน้า reports + approve flow จะใส่ภายหลัง]

### 3.2 ตามสาขาที่ยังไม่ส่ง

**ทำอะไร:** เช็ค deadline · เร่งสาขาที่ค้าง

1. เข้า `/cashhub/reports`
2. ดู section "⚠️ ยังไม่ส่ง" — ระบบบอกว่าสาขาไหนค้างกี่วัน
3. โทรหรือ LINE หาผู้จัดการสาขานั้น
4. ถ้าจำเป็น — กรอกแทนเอง (ดู 3.3)

### 3.3 กรอกแทนสาขา (ถ้าจำเป็น)

**ใครทำได้:**
- `admin` · `area_manager` — กรอกได้ทุกสาขา / ทุกสาขาในเขต (cross-branch access)
- `branch_manager` — เฉพาะสาขาที่ตัวเอง assigned

**ทำอะไร:**

1. เข้า `/cashhub/quick-fill` — เครื่องมือกรอกแทนแบบเร็ว
2. หรือเข้า `/liff/report/[branchId]` ตรง ๆ (เปลี่ยน `[branchId]` เป็น ID สาขา)
3. กรอกข้อมูลตาม form — ระบบจะ render fields ตามประเภทธุรกิจของสาขานั้น (ปั๊ม / เช่า / ฯลฯ)
4. เลือก "วันที่" ที่ต้องการกรอก (ย้อนหลังได้สูงสุด 14 วัน · ยกเว้นวันที่อนุมัติไปแล้ว)
5. กด "ส่งรายงาน"

> ⚠️ ระบบบันทึก audit log ว่าใครกรอกแทน · ใครเป็นเจ้าของรายงานจริง

### 3.4 จัดการเงินขาด

**ทำอะไร:** ดูว่าวันนี้สาขาไหนเงินขาด · ใครรับผิดชอบ

1. เข้า `/cashhub/shortages`
2. ดูรายการเงินขาดของสาขาที่ดูแล
3. กรองตาม: วันที่ / สาขา / คนรับผิดชอบ
4. ถ้ามีเงินขาด → ติดตามกับพนักงานที่ระบุ

[screenshot: หน้า shortages จะใส่ภายหลัง]

### 3.5 ตรวจ Note จากพนักงานสาขา

**ทำอะไร:** ดูข้อความ / ปัญหาที่พนักงานเขียนไว้กับรายงาน

1. เข้า `/cashhub/notes`
2. ดูรายการ note ของสาขาที่ดูแล
3. กดเข้าดู — มี note อะไรน่าสนใจ?
4. ตอบกลับใน LINE หรือ ตามไปที่สาขา

## 📅 4. งานประจำสัปดาห์ / เดือน

### 4.1 ดู Leaderboard สาขา (ทุกวันจันทร์)

**ทำอะไร:** เทียบสาขาของคุณกับสาขาอื่น

1. เข้า `/cashhub/leaderboard`
2. กรองด้วย business type ที่คุณดูแล
3. ดู Health Score (A–F) · streak · ยอดรวม
4. สาขาที่ตกอันดับ — คุยกับ ผจก.สาขานั้น

### 4.2 ดูภาพรวมสาขาที่ดูแล

**ทำอะไร:** เห็น dashboard เฉพาะสาขาของคุณ

1. เข้า `/cashhub/my-branches`
2. เห็นยอด · target · forecast เฉพาะสาขาที่ assigned
3. กดเข้าสาขาแต่ละแห่ง → `/cashhub/branches/[id]` → ดู 30 วันย้อนหลัง

### 4.3 ดู Calendar Heatmap

**ทำอะไร:** ตรวจว่าสาขาส่ง / ไม่ส่งวันไหนบ้าง (visual)

1. เข้า `/cashhub/heatmap`
2. เห็น matrix สาขา × วัน — สีแสดงสถานะ (สีเขียว = ส่ง · สีแดง = ขาด)
3. ระบุ pattern — เช่น สาขา A ขาดทุกวันอาทิตย์ → ต้องตั้ง alarm

[screenshot: หน้า heatmap จะใส่ภายหลัง]

### 4.4 ดู Pattern Heatmap

**ทำอะไร:** ดู pattern เวลาส่งรายงาน (สาขาส่งช้า / เร็ว)

1. เข้า `/cashhub/dashboard` → scroll ไปดู Pattern Heatmap
2. เห็นว่าแต่ละสาขาส่งเวลาไหน
3. สาขาส่งช้าเกิน deadline บ่อย → ต้องเตือน

### 4.5 จัดการพนักงานในสาขา (เฉพาะ `admin`)

> เฉพาะตำแหน่ง `admin` · `area_manager` กับ `branch_manager` ทำไม่ได้

1. เข้า `/users`
2. กดปุ่ม **เชิญผู้ใช้**
3. กรอกข้อมูล + เลือกบทบาท (ปกติ = `staff` หรือ `branch_manager`)
4. assign สาขาที่ต้องดูแล
5. ส่งลิงก์เชิญให้พนักงาน

> 💡 ถ้าพนักงานลาออก · ตอนนี้ deactivate ยังไม่มี UI · ต้องบอก Admin บริษัท

### 4.6 ดูสาขา GPS / ข้อมูล (เฉพาะ `admin`)

1. เข้า `/branches`
2. ดูรายการสาขา · GPS · ผู้จัดการ · deadline
3. กดเข้าสาขา → แก้ไขข้อมูลได้ (เฉพาะ admin)

### 4.7 เอกสาร DocuFlow

**ทำอะไร:** ตรวจเอกสารใกล้หมดอายุของสาขา

1. เข้า `/docuflow/expiry`
2. ดูเอกสารใกล้หมด → กดเข้า → ต่ออายุ
3. รถสาขา (ทะเบียน · ป.วินิจฉัย) → เข้า `/docuflow/vehicles`
4. คนสาขา (ใบขับขี่ · บัตรประชาชน) → เข้า `/docuflow/persons`

## ❓ 5. ปัญหาที่เจอบ่อย + วิธีแก้

**Q: เข้าได้แต่ไม่เห็นสาขาที่ควรดูแลเลย**
A: Admin บริษัทลืม assign สาขาให้คุณ · บอก Admin ให้เข้า `/users` แล้ว assign สาขาให้

**Q: พนักงานสาขาบอกว่ากรอกไม่ได้ — ทำไง**
A:
1. เช็คว่าพนักงานคนนั้น assigned สาขานั้นหรือเปล่า (เข้า `/users`)
2. เช็คว่าวันนั้นยังไม่อนุมัติ (อนุมัติแล้ว = ล็อก แก้ไม่ได้)
3. เช็คว่า browser พนักงาน online · network โอเค

**Q: ส่งรายงานแล้วแต่อยากแก้**
A:
- ถ้า status = "ร่าง" หรือ "ไม่อนุมัติ" → พนักงานแก้ได้เอง
- ถ้า status = "รออนุมัติ" → กด "ไม่อนุมัติ" ก่อน · พนักงานแก้ · ส่งใหม่
- ถ้า status = "อนุมัติแล้ว" → ต้องให้ super_admin ปลดล็อก (คุณทำไม่ได้)

**Q: Quick Approve Bar ไม่ขึ้น**
A: ต้องเลือก checkbox อย่างน้อย 1 รายการก่อน Quick Approve Bar ถึงจะแสดง

**Q: Dashboard ช้า · มีสาขาเยอะ**
A: ลองกรอง business type หรือกรองเฉพาะสาขาที่ดูแล · ระบบ optimize เพิ่มอยู่

**Q: เห็น Health Score แล้ว — มันคำนวณยังไง**
A: A–F ดูจาก: ส่งครบทุกวันหรือเปล่า (40%) · ส่งทันเวลา (20%) · ความถูกต้อง / ไม่ขาด (20%) · streak ติดต่อกัน (20%) · ดูสูตรละเอียดที่ `/cashhub/branches/[id]` ใต้ section "Health Breakdown"

**Q: ทำไมยอดเมื่อวานไม่โชว์ใน "เมื่อวาน ฿X" ของหน้าพนักงาน**
A: เพราะรายงานเมื่อวานยังไม่ได้อนุมัติ · ระบบโชว์เฉพาะรายงานที่อนุมัติแล้วเป็น reference

**Q: รายงาน 1 วันมี 2 ใบ (กะเช้า + กะเย็น) — ทำไม**
A: บางสาขามี shift (morning / midday / evening) · พนักงานเลือกตอนกรอก · ระบบรวมยอดให้ใน Dashboard

**Q: เปลี่ยนรหัสผ่านตัวเองยังไง**
A: เข้า `/profile` → "เปลี่ยนรหัสผ่าน" → กรอกรหัสเก่า + รหัสใหม่

**Q: ออกจากระบบ**
A: เร็ว ๆ นี้จะมีปุ่ม Logout ใน navbar (ตอนนี้ยังพัฒนา) · ระหว่างนี้ปิด browser tab + clear cookie

## 💡 8. Tips สำหรับ Branch Manager / Admin

1. **เช็คงานเช้า 5 นาที** — เปิด `/home` ดูรายงานรออนุมัติ · อนุมัติทันที กันค้าง
2. **อย่ารออนุมัติทีเดียวสิ้นเดือน** — ตรวจรายวัน · ผิดอะไรจับได้ทัน · พนักงานยังจำได้
3. **ใช้ Quick Approve Bar** — ถ้ามั่นใจหลายใบ · ประหยัดเวลา
4. **ตั้ง LINE Notification ของ branch** — ใส่ line_group_id ในข้อมูลสาขา (เร็ว ๆ นี้จะเชื่อม bot)
5. **คุยกับสาขาที่ Health Score ต่ำกว่า C** — มีปัญหาแน่ · อย่ารอ
6. **กรอกแทนเป็นทางเลือกสุดท้าย** — ฝึกพนักงานให้กรอกเองดีกว่า · มี audit log ตรวจสอบได้
7. **ใช้ Heatmap หา pattern** — สาขาที่ขาดทุกวันอาทิตย์ = ไม่มีคนทำงานวันนั้น
8. **DocuFlow Expiry เช็คทุกสัปดาห์** — เอกสารหมดอายุ = ปัญหาใหญ่ในวงจรปั๊ม
9. **อ่าน Notes พนักงาน** — เจอ insight ที่ admin ไม่รู้ (เช่น "ลูกค้าบ่นน้ำมันลด")
10. **อย่าใช้บัญชีเดียวร่วมหลายคน** — ทุก action บันทึก audit · จะแยกคนผิดไม่ออก
