# คู่มือใช้งาน Pooilgroup ERP

> ระบบจัดการธุรกิจน้ำมัน สำหรับสาขา · สำนักงาน · ผู้บริหาร
> Last updated: 2026-05-20
> Production URL: [pooilgroup.vercel.app](https://pooilgroup.vercel.app)

---

## 👋 ยินดีต้อนรับ

Pooilgroup ERP คือระบบบริหารธุรกิจน้ำมันที่ช่วยให้คุณ:

- 📊 **CashHub** — เก็บยอดขายรายวันของทุกสาขา · ตรวจเงินขาด · อนุมัติรายงาน
- ⛽ **FuelOS** — บริหารราคาน้ำมันขายส่ง · ลูกค้า B2B · ทีมขาย (อยู่ระหว่างพัฒนา)
- 📄 **DocuFlow** — เก็บเอกสารบริษัท · แจ้งเตือนใกล้หมดอายุ · เซ็นเอกสารออนไลน์

ระบบเปิดใช้บนเว็บ + มือถือ · ไม่ต้องติดตั้งโปรแกรม · เปิดผ่าน browser เช่น Chrome / Safari ได้เลย

---

## 🧭 หาคู่มือของคุณ

คุณมีหน้าที่อะไรในบริษัท? เลือกคู่มือที่ตรงกับงานคุณ:

### 👑 [คู่มือ Owner / ผู้บริหารระดับสูง](./owner.md)

**สำหรับใคร:**

- เจ้าของบริษัท · CEO · กรรมการผู้จัดการ
- ผู้บริหารที่ดูภาพรวมทุกสาขา ทุกบริษัทในเครือ
- ตำแหน่งในระบบ: `super_admin` หรือ `org_admin`

**คุณทำอะไรได้:**

- ดูยอดขายรวมทุกสาขาทั้งเครือ
- ตั้งค่าระบบ · เปิด-ปิด module · ตั้งกฎคุมความเสี่ยง
- เชิญพนักงานเข้าระบบ · กำหนดสิทธิ์
- ปลดล็อกรายงานที่อนุมัติไปแล้ว (สิทธิ์เฉพาะ super_admin)
- ดู audit log ทุกการเคลื่อนไหวในระบบ

### 🏢 [คู่มือ Branch Manager / Admin](./branch-admin.md)

**สำหรับใคร:**

- ผู้จัดการสาขา · หัวหน้าเขต · ผู้ช่วยผู้จัดการ
- พนักงานออฟฟิศที่ดูแลข้อมูลพนักงานและสาขา
- ตำแหน่งในระบบ: `admin` · `area_manager` · `branch_manager`

**คุณทำอะไรได้:**

- ดูยอดขายของสาขาที่คุณดูแล (เขต · cluster · หรือสาขาเดียว)
- อนุมัติ / ไม่อนุมัติรายงานประจำวันจากพนักงานสาขา
- กรอกรายงานแทนสาขาที่ลืมส่ง
- ดู Leaderboard และเงินขาดของแต่ละสาขา
- จัดการพนักงานในสาขาที่ดูแล (ตำแหน่ง admin · area_manager เท่านั้น)

### 👷 [คู่มือพนักงานสาขา](./staff.md)

**สำหรับใคร:**

- พนักงานหน้าสาขา · พนักงานหน้าปั๊ม
- คนกรอกยอดประจำวัน · ใช้มือถือเป็นหลัก
- ตำแหน่งในระบบ: `staff`

**คุณทำอะไรได้:**

- กรอกยอดขายประจำวันของสาขาที่ได้รับมอบหมาย
- ดูสถานะรายงานที่ส่งไปแล้ว (รออนุมัติ / อนุมัติแล้ว / ไม่อนุมัติ)
- ดูเงินขาด-เงินเกินที่บันทึกไว้
- ตรวจ deadline ส่งรายงาน

## ⚡ ขั้นตอนเริ่มต้นใช้งาน (ทุกคนเหมือนกัน)

1. **ผู้ดูแลระบบเชิญคุณ** — คุณจะได้รับลิงก์เชิญ (invite link) ทาง LINE หรืออีเมล
2. **ตั้งรหัสผ่านครั้งแรก** — คลิกลิงก์ → กรอกชื่อ + รหัสผ่าน → เข้าระบบ
3. **เข้าสู่ระบบ** — ใช้ [pooilgroup.vercel.app/login](https://pooilgroup.vercel.app/login) ทุกครั้งต่อจากนี้

> 💡 ถ้าลืมลิงก์เชิญ หรือลิงก์หมดอายุ → ติดต่อ Admin บริษัทคุณให้ส่งมาใหม่

## 🆘 ติดต่อขอความช่วยเหลือ

| ปัญหา | ติดต่อใคร |
|------|-----------|
| เข้าระบบไม่ได้ · ลืมรหัสผ่าน | Admin ของบริษัท (super_admin / org_admin) |
| ใช้งานไม่เป็น · มีคำถาม | อ่านคู่มือก่อน · ถ้ายังไม่เข้าใจ ถาม Admin |
| ระบบขัดข้อง · error | แจ้ง Admin ส่งต่อให้ทีม IT |
| แนะนำ / ขอ feature ใหม่ | แจ้ง Admin · รวบรวมส่ง Owner |

> Admin ของแต่ละบริษัทมีหน้าที่ดูแลคำถามชั้นแรก · ทีม IT จะถูกเรียกเมื่อจำเป็นจริง ๆ

## 🔐 หลักความปลอดภัย (ทุกคนควรรู้)

- **ห้ามส่งรหัสผ่านให้คนอื่น** · ไม่ว่าเหตุผลใด
- **Admin จะไม่ขอรหัสผ่านคุณทาง LINE / โทรศัพท์** · ถ้ามีคนถาม = ปลอม
- **ออกจากระบบทุกครั้งที่ใช้เครื่องคนอื่น** · กดปุ่ม Logout
- **ระบบล็อก auto** หลังไม่ใช้ 60 นาที (อยู่ระหว่างเปิดใช้)
- **ทุกการกระทำถูกบันทึกใน audit log** · ใครเข้า · ใครแก้ · ใครอนุมัติ · เห็นย้อนหลังได้ทั้งหมด

## 📦 module ที่ใช้ได้จริงตอนนี้

ตามสถานะปัจจุบัน (อ้างอิง STATUS.md):

| Module | สถานะ | ใช้ได้แค่ไหน |
|--------|--------|-------------|
| 📊 CashHub | ✅ พร้อมใช้ | กรอกยอด · อนุมัติ · Dashboard · Leaderboard · Health Score · เงินขาด ครบ |
| 📄 DocuFlow | ✅ พร้อมใช้ | จัดเก็บเอกสาร · แจ้งเตือนหมดอายุ · จัดการรถ/คน ครบ (External Sign ยังไม่ทำ) |
| ⛽ FuelOS | 🟡 บางส่วน | Schema เสร็จแล้ว · UI Price Engine กำลังพัฒนา · ยังไม่เปิดใช้จริง |
| 🤖 LINE Bot / Telegram Bot | ⬜ ยังไม่เปิด | ยังต้องเข้าเว็บเพื่ออนุมัติทุกอย่าง |

> หาก feature ใดยังไม่อยู่ในคู่มือ — ส่วนใหญ่เพราะยังไม่เปิดใช้งานจริง

## 🗺️ Roadmap ใกล้ ๆ (สั้น ๆ)

ที่กำลังทำ / ใกล้ใช้งานได้:

1. **FuelOS Price Engine** — ตั้งราคาน้ำมันขายส่งรายวัน + Margin ตามโซน

4. **External Sign** — เซ็นเอกสารผ่าน OTP (ไม่ต้องมีบัญชี Pooilgroup)
