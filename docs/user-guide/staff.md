## 📌 1. ภาพรวม

คุณคือ "คนต้นน้ำ" ของระบบ — ข้อมูลที่คุณกรอกทุกวัน คือสิ่งที่ Owner และ Manager ใช้ตัดสินใจ

ในมุมระบบ คุณมีหน้าที่:

- **กรอกยอดขายประจำวัน** ของสาขาที่ได้รับมอบหมาย (ก่อน deadline ทุกวัน)
- **ตรวจเงินสด** ตอนปิดกะ — ถ้าเงินขาด ระบุคน + เหตุผล
- **ดูสถานะรายงาน** ของตัวเอง (รออนุมัติ / อนุมัติแล้ว / ไม่อนุมัติ)
- **เขียน Note** ฝากเรื่องไปหา Manager ถ้ามีปัญหา

ระบบใช้บนมือถือเป็นหลัก · ใช้ผ่าน browser (Chrome / Safari) เปิดที่ลิงก์ที่ Manager ส่งให้

## 🚦 2. เข้าระบบ

### ครั้งแรก

1. Manager / Admin ส่งลิงก์เชิญมาทาง LINE
2. กดลิงก์ → กรอกชื่อ + ตั้งรหัสผ่าน → กด "ยืนยัน"
3. ระบบพาเข้าหน้าแรก

### ครั้งต่อไป

วิธีที่ 1 (แนะนำ — เร็วที่สุด): **กดลิงก์สาขาที่ Manager ส่งให้**
- ลิงก์จะอยู่ในรูป `pooilgroup.vercel.app/liff/report/[รหัสสาขา]`
- เก็บไว้ใน LINE chat กับ Manager หรือ bookmark ในมือถือ

วิธีที่ 2: เข้าเว็บแล้วเลือกสาขา
1. เปิด [pooilgroup.vercel.app/login](https://pooilgroup.vercel.app/login)
2. กรอกอีเมล + รหัสผ่าน → กด "เข้าสู่ระบบ"
3. ระบบพาเข้า `/home` → กดสาขาที่จะกรอก

[screenshot: หน้า login จะใส่ภายหลัง]

> 💡 **Tip:** add ลิงก์สาขาเข้า "หน้า Home" ของมือถือ (Safari → Share → Add to Home Screen) จะเปิดเร็วเหมือนแอป

### ⚠️ เข้าไม่ได้ทำยังไง

| อาการ | วิธีแก้ |
|------|---------|
| ลืมรหัสผ่าน | กด "ลืมรหัสผ่าน" → ระบบส่งลิงก์ตั้งใหม่ทาง LINE/อีเมล |
| ลิงก์เชิญหมดอายุ | บอก Manager ให้ส่งลิงก์ใหม่ |
| เข้าได้แต่ไม่เห็นสาขา | Manager ลืม assign · แจ้ง Manager |
| กรอกแล้วบอก "ไม่มีสิทธิ์" | สาขานี้ไม่ใช่ของคุณ · เช็คว่าใช้ลิงก์ของสาขาตัวเอง |

## ✅ 3. งานประจำวัน (Daily Tasks)

### 3.1 กรอกยอดขายปลายกะ

**เมื่อไหร่:** ตอนปิดกะ · หรือก่อน **deadline ของสาขา** (default 21:00 — ดูในหน้ากรอกจะมี countdown)

**ขั้นตอน:**

1. กดลิงก์สาขาที่ Manager ส่งมา (เช่น `pooilgroup.vercel.app/liff/report/abc123`)
2. ระบบโชว์หน้ากรอกของสาขาคุณ — fields ขึ้นอัตโนมัติตามประเภทธุรกิจ (ปั๊ม / เช่า / ฯลฯ)
3. ดูบน header:
   - **ชื่อสาขา + วันที่** (default = วันนี้)
   - **Countdown** เหลือเวลากี่ชั่วโมงก่อน deadline (สีแดงถ้าเลยแล้ว)
   - **"เมื่อวาน ฿X"** — ยอดเมื่อวานให้ดูเปรียบเทียบ (ไม่ auto-fill)
   - **🔥 N วัน** — streak ส่งติดต่อกัน
4. กรอกตัวเลขทุกช่อง:
   - ยอดเงินสด
   - ยอดบัตร / QR / โอน
   - น้ำมันที่ขาย (ลิตร) สำหรับสาขาปั๊ม
   - ค่าเช่า / รายได้อื่น (ถ้ามี)
5. ดู **Reconcile indicator** — ถ้า ✅ เขียว = ยอดตรง · ถ้า ⚠️ แดง = ยอดไม่ตรง (ดู 3.2)
6. กดปุ่ม **"ส่งรายงาน"** (สีน้ำเงิน · ล่างสุด)
7. รอ Manager อนุมัติ — สถานะเปลี่ยนเป็น "รออนุมัติ"

[screenshot: หน้ากรอกยอดประจำวัน จะใส่ภายหลัง]

> 💡 **Auto-save:** ระบบ save ข้อมูลใน browser อัตโนมัติ · ถ้าเผลอปิดมือถือ กลับเข้ามาใหม่ ข้อมูลยังอยู่

### 3.2 ถ้าเงินขาด / เงินเกิน

**เมื่อไหร่:** Reconcile indicator แสดง ⚠️ แดง

**ขั้นตอน:**

1. ระบบเปิด **Shortage Modal** อัตโนมัติ
2. กรอก:
   - **จำนวนเงินที่ขาด/เกิน** (บาท)
   - **คนรับผิดชอบ** (ถ้าระบุได้)
   - **เหตุผล** (เช่น "ลูกค้าจ่ายไม่ครบ" "เครื่อง POS ขัดข้อง")
3. กด "ยืนยัน"
4. ส่งรายงานต่อ

> ⚠️ **อย่าโกหก** ระบุเหตุผลจริง · เพราะเป็นหลักฐานในระบบ · audit ตรวจได้

### 3.3 กรอกย้อนหลัง (ถ้าลืมกรอกเมื่อวาน)

**ทำได้:** ย้อนหลังได้สูงสุด **14 วัน** · ยกเว้นวันที่ Manager อนุมัติไปแล้ว (ล็อก)

**ขั้นตอน:**

1. เข้าหน้ากรอกของสาขา
2. กด dropdown "วันที่" — เลือกวันที่ลืม
3. ระบบโชว์สถานะแต่ละวัน:
   - **เปิด** — ยังไม่กรอก · กรอกได้
   - **รออนุมัติ** — กรอกแล้ว · แก้ได้
   - **อนุมัติแล้ว** — ล็อก · แก้ไม่ได้ (ต้องให้ super_admin ปลดล็อก)
4. กรอกเหมือนปกติ → ส่ง

### 3.4 ดูสถานะรายงานที่ส่งไปแล้ว

**ทำอะไร:** เช็คว่ารายงานคุณอนุมัติแล้วหรือยัง

1. เข้า `/home` หลัง login
2. ดู section "รายงานล่าสุด" — เห็น 7 วันย้อนหลังที่คุณส่ง
3. สถานะแต่ละใบ:
   - ⏳ **รออนุมัติ** (สีเหลือง) — Manager ยังไม่ดู
   - ✅ **อนุมัติแล้ว** (สีเขียว) — เสร็จเรียบร้อย
   - ❌ **ไม่อนุมัติ** (สีแดง) — Manager ขอให้แก้ · กดเข้าดูเหตุผล แก้แล้วส่งใหม่
   - 📝 **ร่าง** (สีเทา) — บันทึกไว้ ยังไม่ส่ง

[screenshot: หน้า home staff จะใส่ภายหลัง]

### 3.5 ถ้ารายงานถูก "ไม่อนุมัติ"

1. กดเข้ารายงานนั้น
2. อ่านเหตุผลที่ Manager เขียน
3. แก้ข้อมูลให้ถูก
4. กด "ส่งใหม่"

> 💡 ถ้าไม่เข้าใจเหตุผล → ถาม Manager ใน LINE ก่อนแก้

## 📅 4. งานประจำสัปดาห์ / เดือน

### 4.1 ตรวจ streak ตัวเอง

**ทำอะไร:** ดูว่าส่งติดต่อกันได้กี่วัน

- 🔥 streak จะโชว์บน header หน้ากรอก
- ส่งครบทุกวัน = streak เพิ่ม · ขาดวันหนึ่ง = streak reset
- streak สูง = Health Score ของสาขาดี = Manager ชม

### 4.2 ดูยอดสะสมเดือนนี้ (ถ้าระบบเปิดให้ดู)

- เข้า `/home` → scroll ดูยอดที่ส่งไปแล้วในเดือน
- เห็นแนวโน้ม — เดือนนี้ขายดีกว่าเดือนก่อนไหม

### 4.3 เปลี่ยนรหัสผ่านทุก 3 เดือน (แนะนำ)

1. เข้า `/profile`
2. กด "เปลี่ยนรหัสผ่าน"
3. กรอกรหัสเก่า + ใหม่ → ยืนยัน

## ❓ 5. ปัญหาที่เจอบ่อย + วิธีแก้

**Q: กดส่งแล้วบอก error · ส่งไม่ได้**
A:
1. เช็ค network — เปิด Google ได้ไหม
2. ลอง refresh หน้า (ข้อมูลใน form ยังอยู่เพราะ auto-save)
3. ลองส่งอีกครั้ง
4. ยังไม่ได้ → screenshot error → ส่งให้ Manager

**Q: ยอดที่กรอกไม่ตรงกับเงินจริง**
A: ใช้ Shortage Modal กรอกส่วนต่าง · ระบุเหตุผลจริง · อย่าใส่เลขมั่ว

**Q: ลืมส่งเมื่อวาน · กรอกย้อนหลังได้ไหม**
A: ได้ · ย้อนหลังได้ 14 วัน (ดูข้อ 3.3) · แต่ Manager จะเห็นว่าส่งช้า

**Q: รายงานวันนี้กรอกแล้ว แต่อยากแก้**
A:
- ถ้าสถานะ "รออนุมัติ" → เปิดอีกครั้ง แก้ · ส่งใหม่ (replace ของเดิม)
- ถ้าสถานะ "อนุมัติแล้ว" → ต้องบอก Manager · เฉพาะ super_admin ปลดล็อกได้

**Q: เห็นว่ามี shift (กะเช้า · กลางวัน · เย็น)**
A: บางสาขาแบ่งกะ · เลือก shift ก่อนกรอก · 1 วันมีหลายใบได้

**Q: ลิงก์สาขาที่เก็บไว้ใช้ไม่ได้แล้ว**
A: อาจเปลี่ยนรหัสสาขา · บอก Manager ส่งลิงก์ใหม่

**Q: มี 2 สาขาที่ดูแล**
A: เข้า `/home` → จะเห็นทั้ง 2 สาขา · กดเลือกสาขาที่จะกรอก

**Q: เครื่องเก่าใช้ไม่ได้แล้ว · จะเปลี่ยนมือถือ**
A: แค่ login ในเครื่องใหม่ · ข้อมูล cloud ทั้งหมด · ไม่หาย

**Q: เห็นรายงานสาขาเพื่อนได้ไหม**
A: ไม่ได้ · ระบบ filter ตามสิทธิ์ · staff เห็นเฉพาะสาขาที่ assigned

**Q: ออกจากระบบ (logout) ยังไง**
A: เร็ว ๆ นี้จะมีปุ่ม Logout · ตอนนี้ปิด browser tab ก็พอ · ระบบจะ auto-timeout เอง

**Q: ทำไม Reconcile แดงทั้งที่ยอดถูก**
A: ลองเช็คทุก field ให้ครบ · บางช่องที่ลืมกรอกจะทำให้สูตรไม่ตรง · ถ้ายังแดงอยู่ ให้กรอกใน Shortage Modal

**Q: มือถือเก่า · ระบบช้า · ค้าง**
A: ลอง:
1. ปิดแอปอื่นในมือถือ
2. ลบ cache browser
3. ใช้ Chrome แทน Safari (หรือกลับกัน)
4. ยังช้าให้แจ้ง Manager

## 💡 8. Tips สำหรับพนักงานสาขา

1. **กรอกก่อน deadline** — ส่งทัน = streak ไม่ขาด = สาขาคะแนนดี = หัวหน้าชม
2. **กรอกทันที** เมื่อปิดกะ · อย่ารอกลับบ้านแล้วค่อยกรอก (มักลืมเลขจริง)
3. **เช็คเลขก่อนกด "ส่ง"** — ถ้าผิดต้องบอก Manager แก้ ยุ่งกว่า
4. **ใช้ Shortage Modal ตรง ๆ** ถ้าเงินขาด · ระบุเหตุผลจริง · ไม่มีใครโกรธถ้าซื่อสัตย์
5. **เพิ่มลิงก์สาขาเข้า Home Screen** ของมือถือ — เปิดเร็ว เหมือนแอปจริง
6. **Bookmark ใน LINE** ลิงก์ที่ Manager ส่งให้ — จะได้ไม่หาย
7. **ใช้ Wi-Fi สาขา** ถ้ามี · เน็ตมือถือบางครั้งหลุดตอนกด ส่ง
8. **อย่า share รหัสผ่าน** กับเพื่อนพนักงาน · ทุก action ระบบจำว่าใครทำ — ถ้ามีปัญหา คุณรับเอง
9. **ส่งทุกวัน** แม้สาขาปิด · บางสาขาตั้งให้ส่ง 0 เพื่อ confirm
10. **ถ้าไม่แน่ใจ ถาม Manager** ก่อน · ไม่ใช่กรอกมั่ว
