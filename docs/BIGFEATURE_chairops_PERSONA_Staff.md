## 5-task walk (ใช้จริงทีละงาน)

### 1. เก็บเงิน (Collect)
- **กี่แตะ:** เปิด LINE → กดเมนู "💰 เก็บเงิน" → หน้าจอกรอกเลข → พิมพ์ "4500" → ถ่ายรูปถังเหรียญ → กด "ส่ง" = **5 แตะ + 1 พิมพ์**
- **ที่พิมพ์:** ตัวเลข 4 หลักเท่านั้น · ห้ามจุดทศนิยม · ห้ามคอมม่า
- **รูป:** ถังเหรียญในกล้องสด (ไม่ใช่จากแกลเลอรี — หนูถ่ายแล้วลืมล้างเก่า)
- **ผิดพลาด:** ห้างชั้น B1 WiFi ขาดๆ · กด "ส่ง" แล้ว spinner ค้าง 30 วิ · หนูกดซ้ำอีก 3 รอบ · กลัวเงินเข้าซ้ำ 3 รอบ

### 2. ตรวจคลีน (Cleanliness)
- **กี่แตะ:** เปิด LINE → กดเมนู "🧹 ตรวจคลีน" → ถ้าทุกอย่างปกติ กด **"ทุกข้อปกติ · ส่งเลย"** → กดยืนยัน = **3 แตะ · ไม่ต้องพิมพ์**
- ถ้ามีปัญหา: กดทีละข้อ (เก้าอี้สะอาด/พื้นสะอาด/ไฟติด/เสียงปกติ/ใบเสร็จมี/ป้ายแขวน) · กดข้อที่ไม่ผ่าน → เปิดให้ถ่ายรูป
- **ที่ถ่าย:** ปัญหาเฉพาะข้อนั้น เช่น คราบบนเก้าอี้ตัวที่ 2
- **ผิดพลาด:** ถุงมือยางหนีบหน้าจอไม่ติด · หนูต้องถอด · เหรียญหล่น

### 3. แจ้งซ่อม (Damage)
- **กี่แตะ:** เปิด LINE → กดเมนู "🔧 แจ้งซ่อม" → เลือกเก้าอี้ (chip ลิสต์) → เลือกระดับด่วน (ปกติ/เร่ง/ฉุกเฉิน) → พิมพ์อาการ "ไฟไม่ติด" → ถ่ายรูป → กด "ส่ง" → **เห็นรหัส RP-2569-0042 ตัวใหญ่** → กดแชร์ไป LINE หัวหน้า = **6 แตะ + 1 พิมพ์สั้น**
- **ที่ถ่าย:** จุดเสีย เช่น มอเตอร์ที่ไม่ทำงาน
- **ผิดพลาด:** กด "ฉุกเฉิน" บ่อยทั้งที่ไม่ฉุกเฉิน · ขอ confirm sheet "แน่ใจหรือว่าฉุกเฉิน?" ก่อนส่ง
- **ต้องการ:** หน้าจอรหัส RP-2569-XXXX **ตัวใหญ่ๆ** · copy-paste ได้ · ส่ง LINE หัวหน้าทันที

### 4. เบิกของ (Supply request)
- **กี่แตะ:** เปิด LINE → "📦 เบิกของ" → เลือกจาก dropdown (ผ้าเช็ดเก้าอี้/น้ำยา/ถุงดำ/หน้ากาก) → ใส่จำนวน → กด "ส่ง" = **4 แตะ + 1 ตัวเลข**
- **ที่พิมพ์:** เลขจำนวน
- **ผิดพลาด:** dropdown ยาวเกิน · หนูเลื่อนผิด · ขอแบ่งหมวด (ของใช้สิ้นเปลือง / ของซ่อม / อื่นๆ)

### 5. ดูยอดวันนี้ (View today)
- **กี่แตะ:** เปิด LINE → "📊 วันนี้" → เห็นยอดที่ส่งไปแล้ว เก็บ + คลีน + ซ่อม + เบิก ครบ ✓ = **2 แตะ**
- **สำคัญมาก** หนูจะรู้ว่าทำครบหรือไม่ · ไม่ต้องเลื่อน chat 200 บรรทัด

**สถานการณ์ขัด:** มือถือร้อนตอนบ่าย · LINE ช้า · กล้องเปิดไม่ได้ · เสียงในห้างดังลำโพง · มี SMS เด้ง · งานต่ออายุประกัน LINE Pay เด้ง · บางทีคนแก่มาขอใช้เก้าอี้ · หนูรีบกดส่ง · พิมพ์ผิด
